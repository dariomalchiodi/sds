# Media campionaria

````{prf:definition}
:label: def_media_campionaria

Indicato con $X_1, \dots, X_n$ un campione di $n \in \mathbb N$ elementi
estratto da una popolazione $X$, la statistica

```{math}
\overline X = \frac{1}{n} \sum_{i=1}^n X_i
```

prende il nome di _media campionaria_.
````
