Indici di concentrazione¶

In presenza di variabili che rappresentano beni condivisibili in una popolazione, come per esempio la ricchezza, ci si può chiedere quanto la variabile sia equamente distribuita tra gli individui della popolazione, oppure quanto sia concentrata solo su un numero ridotto di osservazioni. Questo concetto è diverso dalla varianza, che misura la dispersione dei valori intorno a un valore medio.

Date nnn osservazioni, indichiamo con a1,…,ana_1, \dots, a_na1​,…,an​ il loro elenco una volta che queste sono state ordinate. Successivamente calcoliamone il valore medio

aˉ=1n∑i=1nai\bar a = \frac{1}{n} \sum_{i=1}^n a_iaˉ=n1​i=1∑n​ai​
(1)

e sommiamole, ottenendo TOT=naˉ=∑i=1nai\mathrm{TOT} = n \bar a = \sum_{i=1}^n a_iTOT=naˉ=∑i=1n​ai​. Possiamo avere due situazioni estreme:

  1. in caso di concentrazione minima tutti gli elementi del campione assumono lo stesso valore: a1=a2=⋯=an=aˉa_1 = a_2 = \dots = a_n = \bar aa1​=a2​=⋯=an​=aˉ;
  2. in caso di concentrazione massima tutti gli elementi del campione assumono il valore 0, a parte uno: a1=a2=⋯=an−1=0a_1 = a_2 = \dots = a_{n-1} = 0a1​=a2​=⋯=an−1​=0 e an=naˉa_n = n \bar aan​=naˉ.

In generale allora ci può interessare valutare un indice di concentrazione, che valga 0 oppure 1 nei casi rispettivamente di concentrazione minima e massima, e che sia negli altri casi sia un valore crescente in funzione della concentrazione. Consideriamo:

  • la frequenza relativa cumulata fino all’iii-esima osservazione: Fi=inF_i = \frac{i}{n}Fi​=ni​, per i=1,…,ni = 1, \dots, ni=1,…,n, e
  • la quantità relativa cumulata fino all’iii-esima osservazione: Qi=∑k=1iakTOTQ_i = \frac{\sum_{k=1}^i a_k}{\mathrm{TOT}}Qi​=TOT∑k=1i​ak​​.

Si verifica facilmente che

  • 0≤Fi≤10 \leq F_i \leq 10≤Fi​≤1 e 0≤Qi≤10 \leq Q_i \leq 10≤Qi​≤1;
  • Qi≤FiQ_i \leq F_iQi​≤Fi​ dal momento che le osservazioni sono state ordinate in modo crescente;
  • Qi=FiQ_i = F_iQi​=Fi​ nel caso di concentrazione minima;
  • Qn=FnQ_n = F_nQn​=Fn​.

Per i=1,…,ni = 1, \dots,ni=1,…,n le coppie (Fi,Qi)(F_i, Q_i)(Fi​,Qi​) indicano che il 100Fi%100 F_i \%100Fi​% della popolazione detiene il 100Qi%100 Q_i \%100Qi​% della quantità considerata. Se si considerano i punti sul piano che sono identificati da queste coppie:

  • nel caso di concentrazione minima tutti i punti (Fi,Qi)(F_i, Q_i)(Fi​,Qi​) giacciono sulla retta F=QF = QF=Q: possiamo dunque dire che in questo caso Fi−Qi=0F_i - Q_i = 0Fi​−Qi​=0 per ogni iii;
import numpy as np
import matplotlib.pyplot as plt
import pandas as pd

plt.style.use('sds.mplstyle')

heroes = pd.read_csv('data/heroes.csv', sep=';', index_col=0)

tot = 10
n = 20
a = [tot/n] * n
q = np.cumsum(a) / tot
f = np.arange(1, n+1) / n
plt.plot([0, 1], [0, 1], linestyle='--', linewidth=2, c='gray')
plt.plot(f, q, 'o')
plt.show()
  • nel caso di concentrazione massima i punti (Fi,Qi)(F_i, Q_i)(Fi​,Qi​) per i=1,…,n−1i = 1, \dots, n-1i=1,…,n−1 giacciono sulla retta Q=0Q = 0Q=0, tranne l’ultimo per cui Fn=Qn=1F_n = Q_n = 1Fn​=Qn​=1: dunque in questo caso Fi−Qi=FiF_i - Q_i = F_iFi​−Qi​=Fi​ per i=1,…,n−1i = 1, \dots, n-1i=1,…,n−1 e Fn−Qn=0F_n - Q_n = 0Fn​−Qn​=0.
tot = 10
n = 20
a = [0] * (n-1) + [tot]
q = np.cumsum(a) / tot
f = np.arange(1, n+1) / n
plt.plot([0, 1], [0, 1], linestyle='--', linewidth=2, c='gray')
plt.plot(f, q, 'o')
plt.show()

Nei casi intermedi si avrà dunque che i punti staranno su una curva “sotto” la retta F=QF = QF=Q, dato che Qi≤FiQ_i \leq F_iQi​≤Fi​, e più tale curva si “allontana” dalla retta, più la concentrazione è alta.

L’area compresa tra la curva dei punti (Fi,Qi)(F_i, Q_i)(Fi​,Qi​) (detta curva di Lorentz) e la retta di equidistribuzione (la retta a 45∘45^{\circ}45∘) è detta area di concentrazione e può essere utilizzata come base per la definizione di appositi rapporti di concentrazione, di cui l’indice di Gini costituisce un esempio. Maggiore infatti è la concentrazione osservata, maggiore sarà tale area.

tot = 10
n = 20
a = [tot/100]*10 + [tot*4/90]*9 + [tot/2]
q = np.cumsum(a) / tot
f = np.arange(1, n+1) / n
plt.fill_between(f, f, q, alpha=0.2)
plt.plot([0, 1], [0, 1], linestyle='--', linewidth=2, c='gray')
plt.plot(f, q, 'o')
plt.show()

Verifichiamo quale sia la concentrazione dell’attributo Strength, nell’idea di verificare se la forza sia più o meno equamente distribuita tra i supereroi.

strength = heroes[pd.notnull(heroes['Strength'])]['Strength']

Le quantità relative cumulate si ottengono ordinando i valori, calcolando le corrispondenti somme cumulate e dividendole per la “quantità” totale di forza.

Q = strength.sort_values().cumsum() / sum(strength)

Le frequenze relative cumulate sono facili da calcolare: il loro vettore ha come componenti i valori che vanno da 1n\frac{1}{n}n1​ a 1, incrementandoli ogni volta di 1n\frac{1}{n}n1​, dove nnn indica il numero di valori osservati.

import numpy as np
n = len(strength)
F = np.arange(1, n+1) / n

Pertanto il grafico che mostra l’area di concentrazione è il seguente, in cui a causa dell’elevato numero di osservazioni è più opportuno visualizzare le quantità relative cumulate (i valori di QQQ) usando una spezzata piuttosto che un insieme di punti.

plt.fill_between(F, F, Q, alpha=0.2)
plt.plot([0, 1], [0, 1], linestyle='--', linewidth=2, c='gray')
plt.plot(F, Q, linewidth=2, c='blue')
plt.show()

Il risultato evidenzia una distribuzione della forza relativamente vicina alla situazione di concentrazione minima.

I diagrammi come quello appena ottenuto devono essere analizzati qualitativamente, mentre spesso è più comodo avere un indice numerico calcolato a partire dai dati, il cui valore possa facilmente essere confrontato con due estremi minimo e massino. In tal senso si definisce indice di concentrazione (o coefficiente) di Gini il rapporto

G=∑i=1n−1Fi−Qi∑i=1n−1FiG = \frac{\sum_{i=1}^{n-1}F_i - Q_i}{\sum_{i=1}^{n-1}F_i}G=∑i=1n−1​Fi​∑i=1n−1​Fi​−Qi​​
(2)

tra la quantità ∑i=1n−1Fi−Qi\sum_{i=1}^{n-1}F_i - Q_i∑i=1n−1​Fi​−Qi​ e il suo valore massimo

∑i=1n−1Fi=1n∑i=1n−1i=1n(n−1)n2=(n−1)2\sum_{i=1}^{n-1}F_i = \frac{1}{n} \sum_{i=1}^{n-1} i = \frac{1}{n} \frac{(n-1)n}{2} = \frac{(n-1)}{2}i=1∑n−1​Fi​=n1​i=1∑n−1​i=n1​2(n−1)n​=2(n−1)​
(3)

Per quanto appena visto,

  • 0≤G≤10 \leq G \leq 10≤G≤1, e
  • G=2n−1∑i=1n−1Fi−QiG = \frac{2}{n-1} \sum_{i=1}^{n-1} F_i - Q_iG=n−12​∑i=1n−1​Fi​−Qi​

Siamo ora in grado di calcolare indice di concentrazione di Gini per l’attributo Strength: sottraendo componente per componente il vettore delle quantità cumulate a quello delle frequenze cumulate, sommando i risultati e moltiplicando per 2n−1\frac{2}{n-1}n−12​ si ottiene il valore dell’indice di concentrazione.

2 * sum(F - Q) / (n-1)
0.41182146354576704

In accordo con il grafico precedentemente generato, GGG assume un valore intermedio tra i due possibili estremi, a indicare una distribuzione della forza debolmente concentrata su un sottoinsieme dei supereroi.
